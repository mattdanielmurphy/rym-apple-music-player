  You can use this page to check which releases you've rated/cataloged could still benefit from submitting media links.
The following streaming services are currently supported:
  • Apple Music
  • Bandcamp
  • Deezer
  • Qobuz
  • SoundCloud
  • Spotify
  • Tidal
  • YouTube
To link this release to a stream available on one of these services, copy/paste the URL or HTML embed code. For Spotify you can also use the "spotify:album:<XXX>" shortcut code.
